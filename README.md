# `@simplifield/prettier-config`

> SimpliField shared [Prettier](https://prettier.io) config.

⚠️ You need Prettier >= 1.17.0

## Usage

**Install**:

```bash
$ yarn add --dev @simplifield/prettier-config
```

**Edit `package.json`**:

```jsonc
{
  // ...
  "prettier": "@simplifield/prettier-config"
}
```
